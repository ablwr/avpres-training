(`{` and `}`) - arrays (things inside of `[` and `]`) - objects ("key": "value" pairs) --- # Other bits to notice - String literals (e.g. "\"Phrase in Double Quotes, inside of Quotes\"") - Quotation marks = strings - Booleans are not wrapped in quotation marks - Elements can repeat but you must use a comma `{ "a":123, "b":[1,2,3], "c": { "a":1, "b":2 } }` --- # Additional Resources - WIP! --- # Learning more - [Computers]({{ site.baseurl }}/presentations/computers.html) - [Metadata]({{ site.baseurl }}/presentations/metadata.html) - [Data wrangling]({{ site.baseurl }}/presentations/data-wrangling.html) - [XML]({{ site.baseurl }}/presentations/xml.html) [Home]({{ site.baseurl}}/)
