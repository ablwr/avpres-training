Access to the source code is a precondition for this. - The freedom to redistribute copies so you can help others (freedom 2). - The freedom to distribute copies of your modified versions to others (freedom 3). By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.
